# 1. 프로젝트소개
>     💊 IoT Medicine Smart Box Service 
>    * 시각장애인과 고령층을 위한 음성 안내 스마트 약 보관함 서비스🔊 

![image](https://user-images.githubusercontent.com/101301915/173757240-997c5293-c3a6-4223-8699-7ad93dcc5523.png)

>      🧑‍🤝‍🧑Value
>* 시각장애로 약을 구분하기 어려운 정보취약계층이 스스로 약의 정보를 확인하여 복용하도록 👴
>* 복지의 사각지대에 놓인 취약계층들을 위한 복지사의 생활보조 효율화 증대🧑‍

>     ❓Why
>* 의약품에 점자표시가 되어져 있지 않으며, 점자가 존재하더라고 시각장애인의 95.9%는 해독이 불가능
>* 1인 고령가구의 증가와 시력저하로 인한 약물 오남용의 위험성 증대
